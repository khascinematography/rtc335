# RTC 335 Cinematography

This repository contains HTML handouts for the **RTC 335 Cinematography** course at **Kadir Has University**.  
It introduces core topics including **exposure**, **f-number (f-stop)**, **depth of field**, and **light metering**.

These materials are provided for educational use only and may not be redistributed without the author’s permission.

## 📁 Project Contents

- `index.html` — entry page linking to all handouts.
- `guide.html` — Exposure Quick Guide.
- `fstop.html` — F-number.
- `dof.html` — Depth of Field and Blur Circles.
- `assets/` — images, fonts/CSS, and a small JS helper for fit-to-screen.
